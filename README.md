## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/fitness-studio-gym.git
```

Go to project directory:

```bash
cd fitness-studio-gym
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
